# ❤️ Heart Disease ML Data Pipeline

A complete Machine Learning data preparation pipeline for the Heart Disease dataset. This project focuses on transforming raw data into a clean, feature-engineered, and model-ready dataset through Exploratory Data Analysis (EDA), data cleaning, preprocessing, and feature engineering.

---

## 📌 Project Overview

The goal of this project is to understand the dataset, identify data quality issues, engineer useful features, and prepare the data for machine learning models.

This project covers the essential preprocessing steps followed in real-world ML workflows before model training.

---

## 🚀 Workflow

* Load and inspect the dataset
* Exploratory Data Analysis (EDA)
* Handle missing values
* Detect and treat outliers
* Data cleaning
* Feature engineering
* Encode categorical features
* Feature scaling (if required)
* Correlation analysis
* Prepare the final model-ready dataset
